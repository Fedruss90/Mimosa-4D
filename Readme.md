Mimosa 4D
A hyperrealistic real-time simulation of a family property in Venturina Terme, Tuscany — built in Unreal Engine 5.
What is this
Mimosa 4D is a personal project: a digital sanctuary designed to preserve a real place in time. The simulation reproduces a villa and its ~3 hectares of land on the Etruscan Coast of Tuscany, with real-time weather data, astronomical sun positioning, seasonal vegetation cycles, and spatial audio layered by time of day and season.
This is not a game. It is a living archive.
Core features (planned)
	•	Photogrammetry-based reconstruction of the property
	•	Real-time weather via Open-Meteo API (coordinates: 43.023°N, 10.605°E)
	•	Astronomical sun and moon positioning via UE5 SunPosition plugin
	•	Seasonal vegetation system with 95+ species
	•	Spatial audio with 115+ fauna species, stratified by season and hour
	•	PC and VR support (Meta Quest 3 / Valve Index via OpenXR)
	•	Multiplayer presence for up to 10 users via Epic Online Services
	•	TV Mode — passive cinematic experience, no input required
Tech stack
	•	Unreal Engine 5.4 — Blueprint only
	•	Lumen (Hardware Ray Tracing) + Nanite + Virtual Shadow Maps
	•	MetaSounds + Steam Audio HRTF
	•	Azure DevOps + GitHub + Git LFS
	•	RealityCapture for photogrammetry pipeline
